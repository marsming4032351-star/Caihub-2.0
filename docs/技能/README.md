# 技能设计

## 当前规划的技能

- 战略拆解技能
- Agent 协同技能
- 多模态采集技能
- 质检规则技能
- 标准编写技能
- 菜单迭代技能
- SOP 审计技能
- 运营诊断技能
- 营销策划技能
- 增长分析技能
- 契约治理技能
- 资产建模技能
- 数据流编排技能

## 原则

- Skill 是可复用执行单元
- Agent 通过 Skill 完成具体任务
- Skill 应绑定输入、输出和适用边界
- Skill 不等于 prompt 名字，应该能映射到稳定能力

- 当前仓库已经具备 Skill 注册表
- 当前主要用于系统蓝图和治理层表达
- 还未形成完整执行运行时
