# campaign_feedback_event.v1

## 定义

营销反馈事件契约，用于记录活动、菜品反馈、经营结果与增长观察。

## Owner

`marketing-agent`

## 典型字段

- `campaign_id`
- `store_id`
- `dish_key`
- `start_time`
- `end_time`
- `traffic_delta`
- `conversion_delta`
- `customer_feedback_summary`
- `content_asset_ref`

## 用途

- 增长分析
- 活动复盘
- 营销策略迭代
- 数据资产沉淀
