# restaurant_data_asset.v1

## 定义

跨域沉淀后的餐饮数据资产契约，用于统一表达菜品、质量、运营、营销与知识结构的聚合结果。

## Owner

`data-asset-agent`

## 典型字段

- `asset_id`
- `asset_type`
- `source_domains`
- `time_range`
- `quality_summary`
- `ops_summary`
- `marketing_summary`
- `knowledge_refs`
- `training_value_score`
- `api_export_ready`

## 用途

- 构建餐饮行业数据资产
- 训练与分析输入
- API 生态输出
- CEO Agent 决策看板
