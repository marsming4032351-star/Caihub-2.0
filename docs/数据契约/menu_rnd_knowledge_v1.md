# menu_rnd_knowledge.v1

## 定义

菜单研发知识契约，用于沉淀菜品标准、研发经验、迭代建议与实验结论。

## Owner

`menu-rnd-agent`

## 典型字段

- `dish_key`
- `version`
- `standard_summary`
- `taste_profile`
- `plating_guideline`
- `iteration_notes`
- `experiment_result`
- `recommended_action`

## 用途

- 菜品标准管理
- 菜单迭代建议
- 研发知识库建设
- CEO Agent 经营摘要输入
